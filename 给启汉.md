## 前提须知

方括号的意思是可选，如`[person]`

尖括号的意思是必须，如`<ip>`

## ssh

作用：用来连接远程服务器

如何使用：

```bash
ssh [<person>@]<ip>
```

## tigervncserver

作用：启动一个桌面环境，等待用户连接

如何使用：

```bash
tigervncserver [-localhost no] :<num>
```

其中，```-localhost no```意思是接受远程连接，在你这里必须

## xtigervncviewer

作用：连接一个vnc桌面服务

其中要注意ip、桌面id即上面的`<num>`

如：

VNC server`127.0.0.1:2`

额外：连接桌面之后，按`F8`可以跳出vncviewer的选项
